Project 06.01.01

Project 06.01.01 is my semestre exam project, this is what i am the most proud of as this is the website I've spent most time on making ever. This website is the conclusion of everything I've learnt this semestre.


Making this website i started sketching and making different ideas as to what a usual portfolio site is. I found out it is a presentation of your self and is a sort of "this is me" website.


I found some different colors i wanted to use and wanted to reflect my favorite colors aswell as my favorite time of the year, summer. This led me to use light, warm, soft colors


I wanted to use the strongest contrast of colors, Yellow and Purple. To show my new knowledge in colors and design i chose to use these two colors as they are opposite on the color wheel


I chose to make my button hovers and text the same nuance as the purple color to make the site more manageable and better to look at, because my background color is a strong contrast


I also chose to use rounded edges in all my CSS to make it more pleasant and less sharp for the eyes.


This project is also where i have ever learnt the most JavaScript as i chose to use my vacation in december to learn as much JavaScript as possible since i had a few struggles last time i used it.


This resulted in me using JavaScript for my menu button and all the sites showing up, as there is only one single HTML file in the website. I decided to make different section tags and then use JavaScript to toggle/fade all the sections whenever you click around the site.


The font used is from Google, and the icons used on my Contact page is used from Font-Awesome, which designs logos and icons for websites, as they will always have the same size as the text.


Overall i am very proud of this site as I've used this project to re-learn most JavaScript using YouTube and W3school and build further on my knowledge in HTML and CSS.
